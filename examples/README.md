# Examples

This folder contains examples and tests for different features.

How to run examples:

```bash
git clone https://github.com/mifi/editly.git
cd editly
npm install
npm run build
cd examples
git clone https://github.com/mifi/editly-assets.git assets
./run commonFeatures.json5
```
